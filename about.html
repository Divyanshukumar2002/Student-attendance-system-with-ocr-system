About the Smart Attendance System

An AI-powered student attendance system using Face Recognition and OCR on Aadhaar & PAN cards

Face Recognition

Real-time face detection and recognition using deep learning. Ensures that only the actual student is present during check-in.

OCR Integration

Extracts important identity information from Aadhaar and PAN cards using Optical Character Recognition (OCR) with Tesseract.

Data Extraction

Aadhaar: Name, Aadhaar Number, DOB, Gender
PAN: Name, PAN Number, Father's Name, DOB

MongoDB Storage

All student data and attendance logs are securely stored in MongoDB for tracking and reporting.

Smart Attendance

Combines face and ID verification to automatically mark attendance with timestamps—preventing proxy entries and ensuring authenticity.
